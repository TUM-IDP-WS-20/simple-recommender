Test first recommender engine

The model used here is trained with following properties:

  • All dataset which has 15751 documents used. It also includes accounting irrelevant data. This is the first test, later we are going to exclude them.
  • Only the first 3 pages of the papers are used in the training
  • Pipeline: Cleaned -> Stopwords Removed -> Lemmatized -> LDA Model -> Similarity Measurement (Cosine) -> Recommendation
  • In total 13191 stopwords are removed. 12854(accounting) + 337(standard english)
  • All words except NOUN', 'ADJ', 'VERB', 'ADV' are removed
  • NUM_OF_TOPICS = 30, NUM_OF_WORDS_IN_EACH_TOPIC = 20 set in LDA model
  • For detail: https://tum-idp-ws-20.github.io/doc/milestone-1-datapreprocessing-steps-comparison-based-on-LDA
